RedNote 검색

중국어 검색어로 영상 게시물을 찾습니다

로그인된 Chrome 사용

쿠팡 검색어가 자동으로 입력됩니다. 필요하면 수정하세요. 영상 검색을 누르면 입력한 내용을 중국어로 번역해 새 Chrome 탭에서 검색합니다.

Chrome에서 찾은 영상만 여기에 표시됩니다.로그인이 풀렸다면 Chrome에서 RedNote에 먼저 로그인해주세요.

미디어 구성

영상 1개 + 대표 이미지 1장

대표 장면 대기
영상을 선택하면 자동으로 대표 장면을 고릅니다.서로 다른 장면을 3~5장 JPG로 저장합니다.

영상 1개와 JPG 이미지 1장을 골라 캐러셀로 게시합니다. 각 종류에서 하나씩만 선택할 수 있습니다.
